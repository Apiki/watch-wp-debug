## Watch WP debug.log
Show notification alerts when debug.log WordPress it's changed

## Dependencies
* [Node.js](https://nodejs.org/en/download/) (>= v6.10.3)

## Usage
```
watch-wp-debug /path/to/file/debug.log
```
